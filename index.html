Run this checklist for every new joiner.
Onboarding ChecklistReset
Add new employee record to Monday.com IT Assets board
Select laptop from available pool — record asset tag and serial number
Print and affix QR asset label to device
Factory reset Windows laptop (clean state before handover)
Employee sets up Windows with personal Microsoft account (first-boot requirement)
Provision Google Workspace account (Gmail / Drive / Calendar / Meet)
Install Chrome — sign in with Google Workspace account as primary profile
Communicate IT Policy: all work files must live in Google Drive, not locally
Add to relevant Shared Drives and Google Groups
Assign Microsoft 365 licence if required (Dynamics / Office apps)
Hand over mouse — and phone / monitor if applicable
Send welcome email to new joiner
Collect all devices and revoke all access before marking complete.
Offboarding ChecklistReset
Confirm last working day with HR / People and Places
Send offboarding email — remind employee to move all files to Google Shared Drive
Collect laptop — verify serial number matches record in Monday.com
Collect mouse, phone, monitors (check Monday.com record for full list)
Confirm all work files transferred to Shared Drives
Suspend Google Workspace account (do NOT delete — keep suspended 30 days)
Remove from all Google Groups and Shared Drives
Sign out all active sessions via Google Admin console
Revoke Microsoft 365 licence if assigned
Factory reset returned laptop
Factory reset returned phone
Update Monday.com record status to Offboarded
Archive Google Workspace data after 30-day suspension period
